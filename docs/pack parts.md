These are the elements within an Edgeware++ pack
index.json -•=•- Is the new .json file that combines captions.json, media.json, prompts.json, web.json. It contains the mood for images, videos, audios, caption, prompts, and links
info.json -•=•- Provides information about the pack itself (i.e, pack name, creator, etc)
discord.dat -•=•- Custom discord status when the setting is enabled
corruption.json -•=•- Corruption settings that dictates how corruption mode behaves through levels
configs.json -•=•- A way for pack makers to set an "intended experience" through the custom config
README.txt file -•=•- Used for disclaimers, putting sources and other information. not really needed but it's a good practice to put one
Icon.ico -•=•- An icon for the pack
loading_splash -•=•- A gif or image that will show up when you start Edgeware
Wallpapers -•=•- Wallpapers for the background of the pack
